# Superstore Sales Data Analysis

## Project Overview
This project analyzes sales data to identify trends, top-performing categories, and regional performance.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- West region has the highest sales.
- South region has the lowest sales.
- Technology category generates the highest revenue.
- Consumer segment contributes the most sales.

## Conclusion
This project demonstrates basic data analysis skills including data cleaning, analysis, and visualization.
